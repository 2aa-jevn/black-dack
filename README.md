# 🖤 BLACK DACK

**Créé pour être unique.**

BLACK DACK est une marque de mode contemporaine créée à Bamako, Mali 🇲🇱, avec une ambition internationale.

## 🎨 Identité de la marque

- **Positionnement** : Luxe accessible, Streetwear premium, Mode contemporaine, Création artisanale
- **Style** : Moody Luxury + Streetwear
- **Couleurs** : 🖤 Noir profond | ⚪ Blanc cassé | 🟡 Doré métallique

## 📦 Collections

### BLACK DENIM
Collection de jeans personnalisés :
- Créations faites main
- Broderies et peintures textiles
- Motifs uniques
- Éditions limitées
- **Produit signature** : BD SIGNATURE JEANS (denim noir premium, détails dorés)

### BLACK WOMAN
Collection féminine :
- Robes élégantes
- Design moderne
- Inspiration africaine contemporaine
